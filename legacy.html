A mathematics genius and computer pioneer, Grace Hopper created computer programming technology that forever changed the flow of information and paved the way for modern data processing.
3 / 5
Grace was the first person to pioneer the idea of an automatic programing. While working on the UNIVAC I and II, she developed the first concept of a compiler. A compiler is a computer program that translates computer code written in one programming language into another language. Before Hopper's compiler, all computer programs were built from scratch-even if they shared much of the same code. The process took a lot of time. Mistakes were often made.
4 / 5
Hopper invented code that we still use today called COBOL. COBOL is code that could be used for businesses and military settings. Hopper wanted people to feel more comfortable using computers, so she made COBOL use words instead of symbols. In a 1980 interview Hopper explained, “What I was after in beginning English language [programming] was to bring another whole group of people able to use the computer easily.” Today, 80% of all daily business dealings around the world use COBOL.

Because of her many incredible accomplishments in her lifetime, Rear Admiral Hopper was the recipient of more than forty honorary degrees, and many scholarships, professorships, awards, and conferences are named in her honor.
