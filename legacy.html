In computing, a compiler is a computer program that translates computer code written in one programming language into another language. Grace was the first person to pioneer the idea of an automatic programing. Before Hopper's compiler all computer programs were built from scratch-even if they shared much of the same code. The process took a lot of time. Mistakes were often made.

Grace has invented the code that we today still use today called COBOL. COBOL is a code that could run on different computers for both businesses and the military. Because the Department of Defense used COBOL, any company that wanted to work with them had to use COBOL too. Today 80 percent of all daily business dealings around the world use COBOL.
